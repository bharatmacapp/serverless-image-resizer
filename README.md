# AWS Serverless Image Resizer

## Project Overview

This project automatically resizes images uploaded to an Amazon S3 bucket using AWS Lambda and Python.

## AWS Services Used

* Amazon S3
* AWS Lambda
* CloudWatch
* IAM

## Features

* Automatic image resizing
* Event-driven architecture
* Serverless implementation
* Cloud-based image processing

## Output

When an image is uploaded to the source S3 bucket, Lambda automatically resizes it and stores the resized image in the destination bucket.
